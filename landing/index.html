Ready to Get Started?

Choose your preferred installation method and start enhancing your Dynamo workflows in minutes.

Package Manager

Install directly through Dynamo's built-in Package Manager. Search for "DynaFetch" and click install.

  1. Open Dynamo 3.0+
  2. Go to Packages ‚Üí Search for Packages
  3. Search "DynaFetch"
  4. Click Install
Open Package Manager

Manual Download

Download the latest release directly from GitHub for offline installation or custom deployment.

  1. Download from GitHub Releases
  2. Extract to Dynamo packages folder
  3. Restart Dynamo
  4. Find DynaFetch in node library
